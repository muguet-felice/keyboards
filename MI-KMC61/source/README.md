# MI-KMC61 Source

QMK / Vial source files for MI-KMC61 firmware.

## Files

- `keyboard.json`  
  QMK keyboard definition.

- `config.h`  
  Keyboard configuration.

- `rules.mk`  
  QMK build options.

- `mi_kmc61.c` / `mi_kmc61.h`  
  Keyboard source files.

- `keymaps/vial/`  
  Vial keymap files.

## Notes

These files are provided for reference and for users who want to build or modify the firmware themselves.

Custom builds, modifications, and build environment setup are not supported.
