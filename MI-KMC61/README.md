# MI-KMC61

MI-KMC61 is a 6-key macro keyboard by Muguet.

## Files

- `MI-KMC61_SetupGuide.pdf`  
  Setup guide and assembly manual.

- `QuickStart.pdf`  
  Package insert note / quick start notice.

- `firmware/`  
  Prebuilt firmware files.

- `source/`  
  QMK / Vial source files.

## Firmware

Use the latest `.hex` file in the `firmware` folder.

## Notes

This product is a DIY keyboard kit.  
Please read the setup guide before assembly, use, or firmware update.
